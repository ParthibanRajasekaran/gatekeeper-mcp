# Code of Conduct

Gatekeeper-MCP is intended to be a respectful, constructive open-source project.

Participants are expected to:

- Be respectful and inclusive.
- Assume good intent.
- Provide actionable feedback.
- Avoid sharing secrets or private company information.
- Keep security discussions responsible and minimal in public issues.

Unacceptable behaviour includes harassment, abusive language, personal attacks, and deliberate disclosure of sensitive information.

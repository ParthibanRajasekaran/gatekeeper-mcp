# One-line Pitch

Gatekeeper-MCP is a zero-trust MCP security layer that audits AI-generated code diffs before commit using repository security and architecture policies.

## Short description

A local policy-as-code guardrail server for AI coding agents.

## Longer description

Gatekeeper-MCP sits between MCP-compatible AI clients and repository changes, validating generated diffs against security, architecture, tenancy, and compliance guardrails before code is committed.

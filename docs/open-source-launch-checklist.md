# Open Source Launch Checklist

Use this checklist before sharing Gatekeeper-MCP publicly.

## Repository readiness

- CI is green.
- CodeQL is green.
- npm audit is green.
- README explains the problem in under 30 seconds.
- SECURITY.md includes threat model and disclosure guidance.
- Demo guide works with MCP inspector.
- Roadmap is clear.

## Evidence readiness

- Add screenshots or a demo GIF.
- Publish a technical article.
- Link the article from README.
- Add repo to GitHub profile pinned repositories.
- Share with AI infrastructure, platform engineering, and security communities.

## Package readiness

- Confirm package name availability on npm.
- Add package-lock.json.
- Add release workflow.
- Publish v0.1.0.

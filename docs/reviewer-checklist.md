# Reviewer Checklist

This checklist is for anyone evaluating Gatekeeper-MCP quickly.

## Two-minute review path

1. Read the README opening.
2. View the architecture diagram.
3. Open SECURITY.md for the threat model.
4. Check demo/run-local-audit.md.
5. Check CI, CodeQL, and npm audit badges.
6. Review docs/roadmap.md.

## Signals demonstrated

- AI infrastructure awareness.
- MCP-native design.
- Security and architecture governance.
- Threat modelling.
- Testability.
- Open-source maintainability.
